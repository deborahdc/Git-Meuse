This sequence of codes format data input from different providers of the Meuse basin into a txt of the same format

Date Discharge
yyyymmdd value

There are additional lines to plot and inspect if there is any strange data

The to_txt_WaterInfoNL.py input data is hourly and based on inspections it had some strange values that needed to be filtered with a give threshold

The to_txt_WL.py has additional steps to calculate the areas based on provided shapefiles
